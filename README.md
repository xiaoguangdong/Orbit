# Orbit / 星筹

Orbit is an Android habit-building app concept. The Chinese name, 星筹, means turning small daily actions into a long-term personal star map.

This repository currently contains the early product and requirements documents for the app.

## Product Direction

Orbit focuses on habit formation rather than full task management. The core experience is simple:

- See today's planned habits.
- Check in with one tap.
- Track streaks, completion rate, and long-term progress.
- Use reminders, groups, and milestones to keep habits grounded in daily life.

The MVP is designed around offline-first Android usage, with local habit records, reminders, streak calculation, and basic insights.

## MVP Scope

- Habit creation, editing, archiving, and deletion.
- Today view with date switching and grouped habits.
- One-tap check-in, undo, skip, and backfill.
- Frequency rules and reminders.
- Habit detail pages with streaks and recent history.
- Basic insights and local persistence.
